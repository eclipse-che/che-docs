= Using URL parameters to select an IDE

Use URL parameters to select your preferred IDE without impacting the IDE selection of other project users.

.Procedure

* Start a workspace by using the factory API and pass in additional URL parameters to load the chosen IDE. For example:
+
----
https://<che_fqdn>#<git_repository_url>?che-editor=_<editor_key>_
----
