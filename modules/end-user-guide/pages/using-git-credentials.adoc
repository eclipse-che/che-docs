= Using Git credentials

As an alternative to the xref:administration-guide:oauth-for-github-gitlab-or-bitbucket.adoc[OAuth for GitHub, GitLab, or Bitbucket] that is configured by the administrator of your organization's {prod-short} instance, you can apply your Git credentials, a xref:git-credentials-store.adoc[credentials store] and xref:git-provider-access-token.adoc[access token] as {kubernetes} Secrets.
