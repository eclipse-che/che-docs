= Selecting an IDE

:context: selecting-an-ide

When you start a {prod-short} workspace, you can use the default IDE, link:https://theia-ide.org/docs/[Eclipse Theia], or select one of the following alternatives:

* JetBrains link:https://www.jetbrains.com/help/idea/getting-started.html[IntelliJ IDEA]
* JetBrains link:https://www.jetbrains.com/help/pycharm/quick-start-guide.html[PyCharm]
* Visual Studio Code - link:https://github.com/Microsoft/vscode/[Open Source ("Code - OSS")]
* Your own IDE defined with the `inline` directive

When starting a workspace, you can select an IDE by using one of the two following options:

* URL parameter `che-editor`
* `.che/che-editor.yaml` file
