= Microsoft Visual Studio Code - Open Source IDE

The {prod-short} build of link:https://github.com/microsoft/vscode[Microsoft Visual Studio Code - Open Source] is the default cloud IDE in a new workspace.

NOTE: Your organization might be using a branded build of this IDE.

You can install extensions for this IDE, as explained in the next section, from the link:https://open-vsx.org/about[Open VSX] registry, which might be xref:administration-guide:extensions-for-microsoft-visual-studio-code-open-source.adoc[configured by the administrator] of your organization's {prod-short} instance.
