= Restoring external PostgreSQL

.Procedure

. Copy the following backups to the PostgreSQL instance:
+
[subs="+quotes,+attributes"]
----
scp -i <path to public key> keycloak.sql  XXXX@XXX.XXX.XXX.XXX:/tmp
scp -i <path to public key> che.sql  XXXX@XXX.XXX.XXX.XXX:/tmp
----

. Restore {identity-provider} database:
+
[subs="+quotes,+attributes"]
----
psql keycloak < /tmp/keycloak.sql
----

. Restore {prod-short} database:
+
[subs="+quotes,+attributes"]
----
psql <database> < /tmp/che.sql <1>
----
<1> {prod-short} server database name
