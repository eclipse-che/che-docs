= Setting up a backup server

This section describes which backup servers supported in {prod-short} and gives hints on how to setup one.

include::partial$snip_internal-backup-server-warning.adoc[]

{prod-short} uses link:https://restic.readthedocs.io/en/latest/[`restic`] tool to manage backup snapshots and to push / pull backup data to / from a backup server.

[NOTE]
====
`restic` is licensed under `BSD 2-Clause License`
====

The following types of backup servers are supported in {prod-short} at the moment:

* `REST`. Developed specifically for `restic`. See link:https://restic.readthedocs.io/en/latest/[how to set up REST server] documentation.
* `Amazon S3` and API compatible alternatives. See link:https://docs.aws.amazon.com/s3/index.html[AWS S3] or your preferred alternative documentation.
* `SFTP` See link:https://access.redhat.com/solutions/2399571[How to configure SFTP server]
ifeval::["{project-context}" == "che"]
or link:https://www.manpagez.com/man/8/sftp-server/[SFTP man page].
endif::[]
