= {prod-short} server

The {prod-short} server is the central service of {prod-short} server-side components. 
It simplifies the usage of the {devworkspace} API in a multitenant context.
It is a Java web service exposing an HTTP REST API. 

{prod-short} server needs access to:

* xref:postgresql.adoc[]
* xref:devfile-registries.adoc[]
* xref:plug-in-registries.adoc[]
* xref:devworkspace-engine.adoc[]
