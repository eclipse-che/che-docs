= Installing {prod-short} in a restricted environment on OpenShift

On an OpenShift cluster operating in a restricted network, public resources are not available.

Deploying {prod-short} and running workspaces requires public resources:

* Operator catalog
* Container images
* Sample projects

Replace these resources by their copy in a registry accessible by the OpenShift cluster.


.Prerequisites
