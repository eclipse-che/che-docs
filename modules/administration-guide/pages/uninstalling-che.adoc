= Uninstalling {prod-short}

WARNING: Uninstalling {prod-short} removes all {prod-short}-related user data!

Use `{orch-cli}` to uninstall the {prod-short} instance.

.Prerequisites

* `{prod-cli}`. See: xref:installing-the-chectl-management-tool.adoc[].

.Procedure

* Remove the {prod-short} instance:
+
[subs="+attributes,quotes"]
----
$ {prod-cli} server:delete
----

[TIP]
====
The `--delete-namespace` option also removes the {prod-short} namespace.

The `--delete-all` option also removes the {devworkspace} Operator, the related resources, and the {prod-short} namespace.
====
