= View workspace logs in {orch-name} console


View {prod-short} workspace logs in the {orch-name} web console.


.Prerequisites
* You have a running {prod-short} workspace.
* You have access to the {orch-name} web console.

.Procedure
. In the {prod-short} dashboard, go to *Workspaces*.

. Click a workspace name to display the workspace overview page.
This page displays the {orch-name} {orch-namespace} name __<{orch-namespace}_name>__.

. Click the upper right *Applications* menu, and click the {orch-name} console link.

. Run the next steps in the {orch-name} console, in the *Administrator* perspective.

. Click *Workloads* > *Pods* to see a list of all the active workspaces.

. In the *Project* drop-down menu, select the __<{orch-namespace}_name>__ {orch-namespace} to narrow the search.

. Click the name of the running pod that runs the workspace.
The *Details* tab contains the list of all containers with additional information.

. Go to the *Logs* tab.

.Verification
* The *Logs* tab displays container logs for the workspace Pod.
